Política de Privacidade

Esta Política de Privacidade descreve como as informações são coletadas, usadas e protegidas ao utilizar o aplicativo de automação de monitoramento de páginas do Facebook.

1. Informações Coletadas

O aplicativo utiliza autenticação via OAuth do Facebook exclusivamente para:

  • Identificar o usuário autenticado
  • Listar páginas do Facebook vinculadas às Contas de Negócio (Business Managers) às quais o usuário possui acesso
  • Monitorar o status de disponibilidade dessas páginas

Nenhuma informação pessoal sensível é coletada além das permissões concedidas pelo próprio usuário.

2. Uso das Informações

As informações obtidas são utilizadas exclusivamente para:

  • Monitorar a disponibilidade de páginas do Facebook
  • Detectar indisponibilidades ou falhas
  • Enviar alertas automatizados para ferramentas externas, como Slack

Os dados não são utilizados para fins publicitários, marketing ou compartilhamento com terceiros.

3. Compartilhamento de Dados

As informações coletadas não são vendidas, alugadas ou compartilhadas com terceiros, exceto quando necessário para o funcionamento do serviço de alertas (por exemplo, envio de mensagens ao Slack).

4. Armazenamento e Segurança

Os dados são armazenados de forma segura e pelo tempo estritamente necessário para a execução da funcionalidade de monitoramento e alertas.

5. Exclusão de Dados

O usuário pode solicitar a exclusão de seus dados a qualquer momento. Para isso, basta entrar em contato pelo e-mail informado abaixo.
